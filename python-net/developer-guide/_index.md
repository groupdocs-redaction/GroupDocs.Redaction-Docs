Welcome to the Developer Guide for GroupDocs.Redaction for Python via .NET. This section provides practical, runnable code examples that show how to use every redaction feature in your own projects. Each example is also available in the companion [GitHub repository](https://github.com/groupdocs-redaction/GroupDocs.Redaction-for-Python-via-.NET).

## Prerequisites

Before you start, review:

- [System Requirements]({{< ref "redaction/python-net/getting-started/system-requirements.md" >}}) — supported platforms, Python versions, and native dependencies.
- [Installation]({{< ref "redaction/python-net/getting-started/installation.md" >}}) — install from PyPI or a pre-downloaded wheel.
- [How to Run Examples]({{< ref "redaction/python-net/getting-started/how-to-run-examples.md" >}}) — set up a project and run any example locally or in Docker.

We recommend [obtaining a temporary license](https://purchase.groupdocs.com/temporary-license/) so you can test every feature without the evaluation-mode limits.

## In this guide

- [Basic Usage]({{< ref "redaction/python-net/developer-guide/basic-usage/_index.md" >}}) — the core redaction operations: text, metadata, image, annotation, spreadsheet, and page redaction, plus reading file information.
- [Advanced Usage]({{< ref "redaction/python-net/developer-guide/advanced-usage/_index.md" >}}) — loading and saving documents, rasterization options, page-scoped redaction filters, reusable redaction policies, and extending the supported-format list.
