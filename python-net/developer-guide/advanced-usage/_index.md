The advanced topics build on the [basic usage]({{< ref "redaction/python-net/developer-guide/basic-usage/_index.md" >}}) operations with control over how documents are loaded and saved, how redactions are scoped to pages, and how rules are packaged for reuse.

## In this section

- [Loading Documents]({{< ref "redaction/python-net/developer-guide/advanced-usage/loading-documents/_index.md" >}}) — load from a local disk or a stream, open password-protected files, and pre-rasterize on load.
- [Saving Documents]({{< ref "redaction/python-net/developer-guide/advanced-usage/saving-documents/_index.md" >}}) — save in the original format, rasterize to PDF, save to a stream, overwrite the source, and tune advanced rasterization options.
- [Using Redaction Filters]({{< ref "redaction/python-net/developer-guide/advanced-usage/using-redaction-filters/_index.md" >}}) — scope a redaction to a page range or a rectangular page area.
- [Create PDF with Image Redaction]({{< ref "redaction/python-net/developer-guide/advanced-usage/create-pdf-with-image-redaction.md" >}}) — produce a rasterized PDF with an image area blacked out.
- [Use Redaction Policies]({{< ref "redaction/python-net/developer-guide/advanced-usage/use-redaction-policies.md" >}}) — describe a reusable set of redactions as a policy and apply it across documents.
- [Extend Supported Extensions List]({{< ref "redaction/python-net/developer-guide/advanced-usage/extend-supported-extensions-list.md" >}}) — register a custom file extension with the engine.
