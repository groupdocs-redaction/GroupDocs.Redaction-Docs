A `Redactor` can open a document from a file path or any readable binary stream, including password-protected files. The topics below show each loading scenario.

## In this section

- [Load from Local Disc]({{< ref "redaction/python-net/developer-guide/advanced-usage/loading-documents/load-from-local-disc.md" >}}) — open a document by its file path.
- [Load from Stream]({{< ref "redaction/python-net/developer-guide/advanced-usage/loading-documents/load-from-stream.md" >}}) — open a document from an in-memory or file stream.
- [Load Password-Protected File]({{< ref "redaction/python-net/developer-guide/advanced-usage/loading-documents/load-password-protected-file.md" >}}) — supply a password through `LoadOptions` to open an encrypted document.
- [Pre-Rasterize]({{< ref "redaction/python-net/developer-guide/advanced-usage/loading-documents/pre-rasterize.md" >}}) — rasterize a document as it is loaded, before applying redactions.
