Perform redactions on your documents with just a few lines of code. The topics below cover the core redaction operations using a local `sample.docx` (the same patterns apply to PDF, Excel, PowerPoint, and image files). Every example is runnable — copy the script, download the linked sample file, and run it.

## In this section

- [Redaction Basics]({{< ref "redaction/python-net/developer-guide/basic-usage/redaction-basics.md" >}}) — apply a single redaction or several at once, and check the redaction result.
- [Text Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/text-redactions.md" >}}) — replace or black out text by exact phrase (case-sensitive or RTL) or regular expression.
- [Metadata Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/metadata-redactions.md" >}}) — erase metadata wholesale or by filter, or rewrite values that match a pattern.
- [Image Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/image-redactions.md" >}}) — black out an image area, clean embedded image metadata, and redact embedded images.
- [Annotation Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/annotation-redactions.md" >}}) — rewrite or delete annotations and comments.
- [Spreadsheet Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/spreadsheet-redactions.md" >}}) — redact content scoped to a worksheet or column.
- [Remove Page Redactions]({{< ref "redaction/python-net/developer-guide/basic-usage/remove-page-redactions.md" >}}) — remove pages, page ranges, or frames.
- [Get File Info]({{< ref "redaction/python-net/developer-guide/basic-usage/get-file-info.md" >}}) — read file type, page count, and size from a path or stream.
- [Get Supported File Formats]({{< ref "redaction/python-net/developer-guide/basic-usage/get-supported-file-formats.md" >}}) — list the formats the engine can process.

For loading, saving, rasterization, policies, and filters, continue to [Advanced Usage]({{< ref "redaction/python-net/developer-guide/advanced-usage/_index.md" >}}).
