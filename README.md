# GroupDocs.Redaction - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Redaction](https://docs.groupdocs.com/redaction/) static documentation.

## About GroupDocs.Redaction

[GroupDocs.Redaction](https://products.groupdocs.com/redaction/) is a document redaction SDK that enables you to redact sensitive information from documents including text, metadata, annotations, and images.

Available for .NET, Java, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/redaction/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
