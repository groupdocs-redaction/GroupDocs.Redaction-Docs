## Java Version Requirements

**Minimum Required:** Java Development Kit (JDK) 8 or later

GroupDocs.Redaction for Java supports the following Java versions:

*   **JDK 8** (1.8) - Minimum supported version
*   **JDK 11** (LTS) - Recommended for production
*   **JDK 17** (LTS) - Recommended for new projects
*   **JDK 21** (LTS) - Latest LTS version
*   Any newer Java version

> **Note:** Java 7 (J2SE 7.0) is no longer supported. Please use JDK 8 or later.

### Verify Java Installation

Check your Java version:

```bash
java -version
```

Ensure `JAVA_HOME` environment variable is set correctly. See [Installation]({{< ref "redaction/java/getting-started/installation.md" >}}) for setup instructions.

## Supported Operating Systems

GroupDocs.Redaction for Java supports any 32-bit or 64-bit operating system that runs the Java runtime environment.

### Windows

*   Windows 10, 11 (x64, x86)
*   Windows Server 2016, 2019, 2022 (x64)
*   Windows 8.1 (x64, x86)
*   Windows 7 (x64, x86) - *Limited support*
*   Microsoft Azure

### Linux

*   Ubuntu 18.04 LTS and later
*   Debian 10 and later
*   CentOS/RHEL 7 and later
*   OpenSUSE Leap 15 and later
*   Other modern Linux distributions with Java runtime

### macOS

*   macOS 10.14 (Mojave) and later
*   macOS 11 (Big Sur) and later
*   macOS 12 (Monterey) and later
*   macOS 13 (Ventura) and later
*   macOS 14 (Sonoma) and later

## Overview

GroupDocs.Redaction for Java does not require any external software or third-party tools to be installed. The library is self-contained and only requires a compatible Java runtime environment.

For installation instructions, see the [Installation]({{< ref "redaction/java/getting-started/installation.md" >}}) guide.
